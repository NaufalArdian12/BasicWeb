position: static;

position static digunakan untuk mengatur element menjadi statis seccara default. Elemen akan mengikuti posisi normal secara default, elemen tidak dipengatuihi oleh properti top, bottom,left, dan right

elem ini diatur menggunakan posisi: static;




position relative

sebuah elemen HTML yang menggunakan position relative akan terletak pada posisi normal. mengatur properti atas, kanan, bawah, dan kiri dari elemen yang diposisikan dengan "relatif" akan membuatnya jauh dari posisi normalnya. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.:

elemen ini diatur menggunakan posisi : relative;



position: fixed

Sebuah element HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan j ika halaman website di scroll. Berlaku pengaturan left, bottom, top dan right pada position fixed

elemen ini diatur menggunakan posisi: fixed
